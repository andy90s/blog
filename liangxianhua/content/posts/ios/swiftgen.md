## 前言

SwiftGen 是一个强大的工具，可以生成静态类型的资源访问代码，包括本地化字符串。

它可以扫描你的 Localizable.strings 文件，并生成一个 Swift 文件，提供类型安全的方式来访问这些字符串。

## 安装

```
brew install swiftgen
```
